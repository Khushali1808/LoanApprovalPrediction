# Loan Approval Prediction App

Predict whether a loan application will be approved or rejected based on basic applicant info.

## Features
- Predict loan approval with sliders.
- Interactive Streamlit frontend.
- REST API using Flask.

## How to Run
1. Install dependencies:
pip install -r requirements.txt
